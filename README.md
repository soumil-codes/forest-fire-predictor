# Forest Fire Predictor 🔥🌲

Forest Fire Predictor is an end-to-end machine learning web application that estimates the Fire Weather Index (FWI) based on environmental factors including temperature, relative humidity, wind speed, rainfall, and other fire weather indicators.

The project follows a complete ML workflow:

- Data preprocessing and feature scaling using Scikit-learn
- Training a Ridge Regression model
- Saving the trained model using Pickle
- Building an interactive web interface using Flask
- Deploying the model as a real-world prediction service

Users can input weather and environmental conditions through the web interface and receive an instant FWI prediction generated by the trained machine learning model.

Tech Stack: Python, Flask, Scikit-learn, Pandas, NumPy, HTML, CSS, Pickle
